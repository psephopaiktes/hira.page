## Overview
Website creation for "Dr.Pedal," a bicycle on-site repair service. The site serves multiple purposes including business partner recruitment, user appeal, and job recruitment, but is organized very concisely. I solidified the site's purpose through interviews and refined the content and wireframe.

## Implementation
Created as a Vue-based SPA. Since current site traffic is not particularly high, I set up the configuration to **host the site for free** except for the domain.

* GitHub
* Netlify
* Netlify Forms

The site is hosted using the free plans of these services.
