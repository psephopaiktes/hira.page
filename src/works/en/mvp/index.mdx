## Overview
A drawing board game that you can play on a smartphone alone if you have paper, pen, and 4 or more players. Based on the rules of [A Fake Artist Goes to New York](https://oinkgames.com/games/a-fake-artist-goes-to-new-york/), a board game, I aimed to create a simple game where everyone can easily have fun together. Supports both Japanese and English.

I handled planning, design, architecture, and implementation all by myself.

## UI
With the expansion of devices in recent times, I adopted a bottom navigation so you don't have to reach your fingers to the top of the display. Of course, it's responsive.
![UI](./ui.png)

## Implementation
Released as a PWA, which allows web pages to be offered as applications. Primarily created with Vue.js. The source code is published on [GitHub](https://github.com/psephopaiktes/mafia-vs-painter).
