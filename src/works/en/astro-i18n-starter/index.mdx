A multilingual website creation theme for use with [Astro](https://astro.build), a frontend environment. I released it as open-source software to make creating multilingual websites simple and easy.

## Features
- Support for Astro's official i18n functionality
- Various management methods for multilingual pages
- Vanilla CSS
- SEO friendly
- 100/100 Lighthouse score
