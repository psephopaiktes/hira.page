## Overview
A very simple Web Components using [CSS mask-image](https://developer.mozilla.org/docs/Web/CSS/mask-image).

SVG images can now be colored with CSS, making them very convenient for using icons.

![Demo Image](./preview.gif)

It has more advantages than icon fonts and is easy to use.
